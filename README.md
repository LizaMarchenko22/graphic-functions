# graphic-functions
Наш сайт — ваш надёжный помощник в изучении функций и графиков для 7–11 классов. Здесь вы найдёте понятные объяснения, примеры и интерактивные графики. Мы поможем разобраться с линейными, квадратичными, тригонометрическими и другими функциями, чтобы подготовка к урокам и экзаменам стала проще и интереснее!
